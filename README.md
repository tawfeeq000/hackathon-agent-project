## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/tawfeeq000/hackathon-agent-project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hackathon-agent-project
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
